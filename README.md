# Vanilla GAN in Tensorflow
This is a very simple step by step implementation of vanilla GAN using Tensorflow.  

### What you can find
* Vanilla GAN
* Xavier Initialization
* Adam optimizer

### Requirements

* Matplotlib (to visualize/save results)
